- Internet cafes
- Wireless hotspot (captive portal)
- VPN router
- Firewall
- DHCP/DNS server
- Wireless access point
- Transparent squid proxy server
- Multi wan router or load balancer
- DNS blacklist
- Port forwarding/NAT (network address translation)
